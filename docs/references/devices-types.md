Devices are pieces of software or hardware used in your project to be controlled from *score*.

*Score* currently support various protocoles as well as special devices types to work with. These are:

- OSCQuery
- OSC
- Minuit
- Mapper
- Local
- Midi in/out
- Serial
- Joystick
- Wiimote
- HTTP
- WS
- Artnet
- Audio
- Window
- Camera input
